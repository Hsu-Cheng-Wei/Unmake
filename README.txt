Unmake aims to allow users to effortlessly transition from the legacy build
system make to more modern and palatable systems, with an emphasis on Visual
Studio and cmake.

For installation instructions for the parser/analyzer on a Unix-like system,
see Unix_Install.txt .

For installation instructions for the project generator on a Windows system,
see ProjGenReadme.txt .

For installation instructions for the web component on a Windows system,
see MCAF - Corporate Intranet Setup.doc .
